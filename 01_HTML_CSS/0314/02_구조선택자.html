02_구조선택자

구조선택자는 부모와 자식요소 태그 관계에서 자식요소들을 선택하고자 할 때 사용하는데, 자식요소들 중에서 첫째, 마지막번째, n번째, 짝수번째, 홀수번째를 찾아서 서식을 적용하고자 할 때 많이 사용한다.

구조선택자의 종류
1. 일반구조 선택자
2. 형태구조 선택자

1. 일반구조 선택자

  • :first-child - 부모요소의 자식요소들 중 첫번째 요소를 선택한다.
  • :nth-child(2) - 부모요소의 자식요소들 중 두번째 요소를 선택한다.
  • :nth-child(3) - 부모요소의 자식요소들 중 세번째 요소를 선택한다.
  • :last-child - 부모요소의 자식요소들 중 마지막번째 요소를 선택한다.

2. 형태구조 선택자

  • :first-of-type - 형태가 다른 자식요소들 중에서 첫번째를 선택한다.
  • :last-of-type - 형태가 다른 자식요소들 중에서 마지막번째를 선택한다.
  • :nth-of-type(n) - 형태가 다른 자식요소들 중에서 n번째를 선택한다.
  • :nth-last-of-type(n) - 형태가 다른 자식요소들 중에서 뒤에서 n번째를 선택한다.
