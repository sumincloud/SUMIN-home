 <h3>테두리 속성 - border</h3>   
    1) border-width : 선의 굵기 지정, px단위 사용, 키워드: thin, thick
   
    2) border-style : 선스타일 지정
      a. solid : 실선                  나) dashed : 긴 점선
      b. dotted : 짧은 점선         라) double : 두줄(3px 이상)
      c. inset : 테두리 위쪽을 어둡게해서 파인듯한 느낌을 주는 선
      d. outset : 테두리 아래쪽을 어둡게해서 위로 튀어올라온 듯한 느낌 주는 선
      e. ridge : 테두리 자체가 튀어올라온 듯한 느낌을 주는 선(3px이상)
      f. groow :  테두리 자체가 파인듯한 느낌을 주는 선(3px이상)
      g. none : 테두리가 적용된 선택자를 선을 없애기 위해 사용
   
    3) border-color : 선의 색상 지정 - color의 값쓰는 방식과 같음
    
    4) 속기법 - border: 굵기 스타일 색상;
   
    5) 위치관련
      a. border-top : 선택자의 위쪽 테두리 지정
      b. border-right : 선택자의 오른쪽 테두리 지정
      c. border-bottom : 선택자의 아래쪽 테두리 지정
      d. border-left : 선택자의 왼쪽 테두리 지정
