04.시멘틱구조태그

시멘틱(sementic)이란?

header : 웹페이지의 상단영역으로서 상단로고, 검색폼, 네비게이션 등이 들어가는 영역
nav : navigation태그의 약자로서 여러개의 메뉴 작성시 사용하는 태그
main : 메인 콘텐츠 영역을 지정하고자 할 때 사용하는 태그
section : 메인 콘텐츠 영역에 작은 독립적인 콘텐츠들을 묶고자 할 때 사용하며 h2-h6요소가 와야 함.
article : 독립적인 콘텐츠 영역을 만들고자 할 때 사용하며, section처럼 h2-h6요소가 나와야 함.
aside : 메인 콘텐츠 영역의 왼쪽/오른쪽 사이트 측면에 배치되는 영역으로서 '퀵메뉴, 최근본상품, 상담원채팅' 등의 콘텐츠 제작 시 사용.
footer : 웹페이지 하단의 푸터영역을 지정하고자 할 때 사용.
