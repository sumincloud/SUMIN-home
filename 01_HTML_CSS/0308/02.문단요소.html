02.문단요소 - p, hr, br

1. p - paragraph(문단)의 약자로서 문단 내용을 작성하고자 할 때 사용하며, 제목태그처럼 자동으로 줄바꿈되는 특성이 있다

2. hr - horizantal(수평선)의 약자로서 가로줄을 표시하고자 할 때 사용하며, 주로 문단내용을 수문하고자 할 때 사용하는 일종의 수평선이다. 현재는 css의 'border'속성으로 사용할 수 있기 때문에 자주 사용되지는 않는다.

3. br - line break의 약자로서 문단의 내용중에 강제로 줄을 변경하고자 할 때 사용하는 태그요소이며, hr태그요소과 같이 닫는


태그가 없는 것이 특징이다.

p, hr태그는 자동으로 줄변경되는 '블럭'요소이다.
