html5 태그 학습하기_01.제목(hn)요소
html문서의 내용을 대표하는 제목을 작성하고자 할 때 사용하는 태그요소이며, heading의 약자인 'h'태그를 사용한다. h태그의 종류에는 h1, h2, h3, h4, h5, h6가 있으며 h1부터 글자가 큰 순서이다.
