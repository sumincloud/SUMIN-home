13_폼 태그

폼 태그란? 사용자가 입력한 정보(아이디, 패스워드 등)를 서버로 전송하는 역할을 하며, 폼 태그를 활용한 양식의 종류에는 로그인폼, 회원가입폼, 검색폼, 예약폼 주문결제 등의 다양한 양식을 만들 수 있다.

폼 태그의 종류
form
input
select, option
textarea
label
button
fieldset
legend

폼 태그 사용하기

폼 필드제목 1. 텍스트박스 : 문자를 입력하기 위한 박스로서 '아이디', '검색', '주소', '문의하기 제목'....


2. 패스워드 박스 : 비밀번호, 비밀번호 확인, 주민번호, 카드번호, 개인정보 작성시 글자가 보이지 않게 '기호'로 표시되게 한다.


3. 라디오 버튼 : 여러개의 항목중에 하나만 선택이 가능한 양식(약관동의, 성별선택, 열차선택, 좌석선택, 시간선택...)


4. 체크박스 : 여러개의 항목들 중에서 사용자가 원하는 항목만 선택할 수 있는 양식
나의 취미 :

5. 선택목록 : 쇼핑몰 옵션선택, 전화번호 입력, 통신사 선택, 예매목록, 인원 수, 언어설정, 패밀리사이트 등의 양식을 만들고자 할 때 사용하는 양식




6. 다중 텍스트 박스 : 한 줄 입력이 아닌 여러 줄의 텍스트 내용을 입력하고자 할 때 사용.(게시판 게시글 작성, 이메일 내용 작성, 댓글작성..)


7. 파일첨부 : 게시판이나 이메일 양식에서 상대방에게 파일을 첨부하여 보내기 위한 양식.

8. 버튼 : 전송하기, 취소하기, 일반버튼, 이미지버튼
