12_멀티미디어 태그

멀티미디어(multimedia) : multi+media, 영상+매체의 합성어로서 다중+매체라고 하며, 책, TV, 라디오, 신문, 잡지 등을 말하며 오늘날에는 인터넷의 비중이 많이 커졌다. 또한 사용자는 PC, TABLET, MOBILE, SMART WATCH, NOTEBOOK 기기를 통하여 정보를 접한다.

웹페이지를 구성하는 요소에는 문자, 이미지, 콘텐츠외에 영상이나, 사운드, 플래시 애니메이션(없어짐) 등이 멀티미디어에 속한다. 영상이나 사운드를 삽입하기 위한 태그요소를 '멀티미디어 태그'라고 한다.

HTML5 이전에서는 object, embed라는 태그를 사용했지만 w3c에서는 html5 웹표준에서 지정한 video, audio태그를 추가하여 표준화하였다.

1. 음악파일의 종류와 삽입방법 - mp3, ogg

음악파일의 확장자는 시스템이나 사용자 환경, 브라우저의 종류, 버전에 따라 지원이 안되는 경우가 있기 때문에 대체할 음악을 미리 넣어주거나 대체 문구를 삽입한다.

위 2개의 음악파일 중 우선되는 음악이 먼저 플레이 된다.

2. 영상파일의 종류와 삽입방법 - mp4, webm, ogv

mp4확장자는 모든 브라우저에서 잘 지원되지만 webm확장자는 익스플로러, 사파리, 안드로이드 브라우저에서는 지원이 안된다. 따라서 webm, ogb확장자 사용시 대체영상을 넣어준다.

  • controls : 컨트롤 창 표시 옵션
  • autoplay : 자동실행
  • muted : 음소거
  • loop : 반복재생

* html태그, css선택자, 속성이 브라우저별 호환성 여부 체크 site : http://caniuse.com
