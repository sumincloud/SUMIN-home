하이퍼링크란? 하이퍼텍스트로 이루어진 문서에서 사용자가 문자를 마우스로 클릭하였을 때 다른 문서나 페이지로 이동하기 위한 기능이다.

3. 이메일 링크

관리자에게 메일 보내기

클릭시 시스템에서 설정된 기본 이메일 프로그램이 실행되며 사용자가 관리자에게 메일을 보내기 위한 방법을 선택하여 메일을 보내게 된다. 해외에서는 많이 사용되는 방법.

4. 특정 위치로 이동되는 네임앵커

현재 페이지에서 메뉴를 클릭시 다른 페이지로 넘어가지 않고, 특정위치로 화면 스크롤이 이동되어 표시되는 기능.
