07.이미지 태그 p.108

웹문서안에 들어가는 이미지의 종류로는 jpg(고화질 사진), png(고화지르 로고, 아이콘), gif(단순한 색상의 이미지나 움직이는 이미지, 아이콘), bmp, webp(구글에서 제작한 이미지 파일 포맷), svg(w3c에서 제작한 벡터방식 이미지)의 종류가 있다.

이러한 이미지 종류들을 웹문서에 삽입하기 위한 태그요소가 img태그이며, source의 약자인'src'속성과 이미지가 나오지 않을 때 대체할 문자인 'alt'속성을 사용한다.

img src = '이미지 파일의 주소나 경로' alt = '이미지 대체문자'

이미지 태그는 닫는 태그가 없는 것이 특징이다.

1. 이미지 삽입하기

상대주소를 사용하여 이미지 삽입하기

절대주소를 사용하여 이미지 삽입하기
url주소
주소를 작성하는 방법에는 1. 절대주소 2. 상대주소가 있다.
1. 절대주소는 http://로 시작하여 파일명.확장자로 끝나는 주소를 말한다.
2. 상대주소는 현재문서를 기준으로 주소가 작성된다.
현재폴더에 이미지 폴더가 있는 경우 - 하위폴더("./폴더명/파일명.확장자")
다른폴더에 이미지 폴더가 있는 경우 - 상위폴더("../폴더명/파일명.확장자")
html문서와 이미지가 같은 위치에 있는 경우 - 동일한 위치("파일명.확장자")

이미지를 대체하기 위한 문자 속성 alternate = alt

alternate text의 약자로서 이미지가 나오지 않을 때 사용자가 이해할 수 있는 대체문자를 말하며, 시각장애인이 스크린리더를 통해서 음성으로 이해할 수 있는 문자이다. 웹접근성을 위하여 반드시 삽입한다.

이미지 크기 조정을 위한 속성 - width, height

이미지 크기값은 이미지의 개수가 많은 경우 width, height속성을 사용하되 이미지의 개수가 적을 경우는 안써도 브라우저에서 자동으로 인식하여 이미지 크기를 표시해준다.

css스타일 시트에서 이미지의 크기를 한꺼번에 통일시켜 설정하면 이미지 빠른 로딩에 도움이 된다.

크기값을 지정할 때 기본단위는 'px', 유동적으로 크기를 변경하고자 할 때는 '%'단위를 사용한다.



svg이미지 삽입하기

svg : Scale Vector Graphic의 약자로서 벡터방식을 사용하여 이미지가 확대되어도 깨지지 않고, 선명하게 나오는 이미지 파일 형식을 말함.

html5에서 새로 추가된 파일 확장자로서 보통 모바일화면에서 화면해상도에 따른 이미지를 깨끗하게 보여주기 위해서 사용하며 로고 등에 활용된다. w3c에서 새롭게 파일 형식을 추가하였다.

svg만들기 : 벡터방식의 그래픽 프로그램인 '일러스트'에서 디자인 작업 후 'file > save as > svg이미지'를 선택하여 저장한다.
