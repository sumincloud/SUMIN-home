12_상태선택자

입력양식의 상태를 선택하고자 할 때 사용하는 선택자이다.

  • :checked - 양식을 체크한 상태
  • :focus - 초점이 맞추어진 상태
  • :enabled - 사용이 가능한 요소를 선택
  • :disabled - 사용이 불가능한 요소를 선택

노랑풍선 '여행지 검색양식', 'id, pw입력양식'을 클릭시 테두리 색상 변경 => focus(초점)

노랑풍선의 로그인 양식아래 '아이디저장' 체크박스 클릭시 아래 내용 나오거나 숨기기, cjmall 고객센터 질문과 답변 top10, 모바일 토글버튼 클릭시 내비게이션 나오거나 숨기기, 본인인증페이지 약관동의시 버튼색 변경 => checked
