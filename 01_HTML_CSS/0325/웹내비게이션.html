웹내비게이션

웹페이지에서 사용자가 정보를 찾기 위한 가장 기본적인 수단으로서 내비게이션(메뉴)라고 하며, 종류에는 메인메뉴, 서브메뉴, 검색메뉴, 로그인메뉴, 사이드 메뉴, 드롭다운메뉴, 탭메뉴, 사용자의 현재 위치를 알려주는 '로케이션', 페이지네이션 메뉴 등이 있다.

내비게이션은 메뉴목록을 나열하기 위해서 목록태그요소 ul, li태그를 사용하며 a태그요소로 하이퍼링크를 설정한다.

내비게이션 명칭
gnb : Global Navigation Bar의 약자로서 웹사이트의 가장 중심에 있는 메뉴를 작성할 때 사용하는 이름으로 메인메뉴라고 하며, 헤더영역에 가독성있게 표시해주는 것이 특징이다.
lnb : Local Navigation Bar의 약자로서 메인메뉴외에 서브메뉴나 콘텐츠 영역에서 사용되는 메뉴 목록을 작성하고자 할 때 사용하는 명칭.
