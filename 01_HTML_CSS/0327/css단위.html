css단위 - pxtoem(사이트)

  • px - pixel, 화면 비율에 상관없이 고정되는 단위(절대단위)
  • % - 부모요소를 기준으로 비율을 표현하는 단위.(상대단위, 반응형에서 주로 많이 사용됨) 예) 16px = 100%, 150% = 24px
  • em - 부모요소의 font-size를 상속받아 크기가 결정되는 단위
  • rem - root의 약자로서 부모의 font-size를 상속받지 않고 최상위(html)요소에서 결정된 font-size를 기준으로 크기가 결정되는 단위. 예) 16px = 1rem
  • vh - viewport height의 약자로서 웹브라우저의 높이를 100으로 기준하여 높이크기가 결정됨.
  • vw - viewport width의 약자로서 웹브라우저의 너비를 100으로 기준하여 가로너비가 결정됨.
  • vmin - 높이와 너비중 짧은 쪽으로 100을 기준하여 크기가 결정됨.
  • vmax - 높이와 너비중 긴 쪽으로 100을 기준하여 크기가 결정됨.

반응형웹에서 값을 자동으로 설정하기 위한 함수
calc(전체값 - 사용할값);
width : calc(전체값 - 사용할값);
margin : calc(전체값 - 사용할값);
