그림자(shadow) 속성 - 텍스트, 박스요소

shadow속성은 문자, 이미지, 블럭요소에 그림자 효과를 적용하고자 할 때 사용하는 속성이다.

  • text-shadow : 문자에 적용할 그림자 효과
  • box-shadow : 박스요소에 적용할 그림자 효과
  • 옵션 : x축(가로), y축(세로), offset(번지는 값), color(색상값)
