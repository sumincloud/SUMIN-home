1. display(가시)속성

html태그는 저마다 보이지 않는 특성을 가지고 있는데 이것을 'display'속성이라고 한다.

display속성의 종류
inline
block
inline-block
table
table-cell
flex
grid
none

블럭방식의 요소 - p, h1~h6, div, header, nav, main, section, article, aside, footer, dl, dt, dd, ul, li, ol, form, fieldset, hr, table

인라인방식의 요소 - img, b, strong, sup, sub, input, label, button, textarea ...

2. boxmodel의 개념

box model이란? 블럭요소와 인라인요소가 가질 수 있는 크기, 마진, 패딩, 테두리 값을 사용하여 배치하는 개념을 말한다.

박스계산공식
전체 가로크기 = 가로너비 + 패딩값 + 마진값 + 테두리값
전체 세로크기 = 세로높이 + 패딩값 + 마진값 + 테두리값

패딩값 주기 : padding : 10px(위, 오른, 아래, 왼쪽 순으로 적용), padding : 0px 10px 20px 30px(12시 방향에서 시계방향으로 돌아가며 값 설정)

마진값도 패딩값 설정방법과 동일함.

100*100
100*100
100*100
100*100
100*100
100*100
100*100
100*100

box-sizing 속성을 사용시 padding, border값은 원래 크기에 포함하여 계산하기 때문에 박스크기에는 변함이 없다.

box-sizing:border-box (padding, border값 포함)

box-sizing:content-box (padding, border값 포함하지 않음)
