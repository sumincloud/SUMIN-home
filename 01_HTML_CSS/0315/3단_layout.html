헤더영역 : 웹페이지의 상단영역으로서 로고, 검색폼, 내비게이션이 들어가는 영역이다.
section : 관련있는 콘텐츠영역을 묶고자 할 때 사용하는 영역으로서 section 사용시 h2~h6요소가 나와야 한다.
article : 독립적인 내용을 작성하고자 할 때 사용하는 영역으로서 section태그와 같이 h2~h6요소가 나와야 한다.
aside : 메인콘텐츠의 왼쪽, 오른쪽 사이드에 배치되는 광고, 최근본상품, 상담원연결 등의 콘텐츠 작성시 사용한다. 고정배치 되는 경우도 있음.
푸터영역 : 웹페이지의 하단영역으로서 하단로고, 회사소개, 이용약관, 회사명, 주소, 연락처, 카피라이트 내용이 들어가는 영역이다.
