선택자(selector)의 종류

  1. 전체 선택자(*)
  2. 클래스 선택자(.class명)
  3. 아이디 선택자(#id명)
  4. 태그 선택자(tag명)
  5. 그룹 선택자(a, b)
  6. 자손, 후손 선택자(a>b, a b)
  7. 링크, 반응 선택자
  8. 구조선택자(일반, 형태)
  9. 동위선택자
  10. 속성선택자
  11. 전, 후문자 선택자
  12. 클래스 결합, 태그와 클래스 결합

05_css선택자_자손_후손선택자
