css속성_background

background속성은 사용자가 선택한 요소의 배경색, 배경이미지를 삽입하기 위한 속성이다.

배경속성의 종류
1. background-color : 배경색(단일색, 그라디언트)
2. background-image : 배경이미지
3. background-repeat : 배경반복설정
4. background-size : 배경이미지 크기설정(px, %)
5. background-position : 배경이미지 위치 지정(x, y)
6. background-origin : 배경이미지 기준위치 지정
7. background-attatchment : 배경이미지 고정하기
8. 멀티백그라운드 : css3에서 추가된 속성으로 여러 이미지를 한꺼번에 배경 삽입가능.

1. 배경색(background-color)적용하기

배경색상을 넣기 위한 속성으로서 색상값의 종류로는 '색상명', hex코드, rgb, rgba, hex, hexa종류가 있다.
