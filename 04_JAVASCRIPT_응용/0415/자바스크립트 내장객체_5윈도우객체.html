자바스크립트 내장객체_5윈도우(window)_브라우저 객체

브라우저 객체는 사용자가 사용하는 웹브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용하는 객체이다.

Window 객체

window객체는 브라우저 객체 중 '최상위 객체'로서 현재 실행되고 있는 브라우저창을 나타낸다.

Window 객체의 속성과 메서드

Window 객체의 속성(property)종류
document
frameElement
innerHeight
innerWidth
location
name
screenX
screenY
scrollX, scrollY
sessionStorge
Window 객체의 메서드(method) 종류
alert(); - 경고창 띄우기
blur(); - 현재창에서 포커스를 제거
close(); - 현재창을 닫기
confirm(); - 확인/취소 버튼이 있는 확인창을 표시
focus(); - 현재창에 포커스를 부여함
moveBy(); - 현재창을 지정한 크기만큼 이동
moveTo(); - 현재창을 지정한 좌표로 이동
open(); - 지정한 문서를 열기
prompt(); - 사용자가 입력가능한 프롬프트창을 활성화함
resizeTo(); - 동적으로 브라우저 창의 크기를 조절
scroll(); - 문서의 특정위치로 스크롤합니다.
sizeToContent(); - 내용에 맞게 창의 크기를 조절
stop(); - 로딩중지
