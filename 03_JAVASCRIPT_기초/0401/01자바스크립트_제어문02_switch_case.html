01자바스크립트_제어문02_switch_case

제어문02_switch_case(선택문)은 사용자가 만든 조건이 만족할 때 실행되고, 프로그램이 그자리에서 종료되는 제어문이다.

선택문은 조건의 값이 정확하게 일치해야 하는게 다른 조건문과 차이점이다.

break문 - 조건식에서 원하는 값이 나왔을 때 프로그램이 정지된다.

    switch (값) {
      case 조건값0 : 출력될값;break;
      case 조건값1 : 출력될값;break;
      default : 출력될값;
    }
